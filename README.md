# earth_photo_manager
写真管理アプリ。簡易現像機能とかも入れたい。

## 実行

```sh
uv run python earth_photo_manager.pyw
```

## ビルド

`.env.sample` を `.env` にコピーして `WUV` を設定したあと、以下を実行します。

```sh
make
```

## 現在の機能

- 複数のルートフォルダ登録
- 登録ルート配下のサブフォルダ閲覧
- 選択中フォルダ直下の画像ファイル一覧
- サムネイル付きファイル一覧
- 別プロセスでバックグラウンド生成されるサムネイルキャッシュ
  - アプリルート内の `.thumbnails/` に保存
  - 未完了分は次回起動後にバックグラウンドで再開
- 同一ウィンドウ内プレビュー
- 別ウィンドウプレビュー切り替え
- 選択中画像のファイル情報表示
- 前回選択していたフォルダ・画像の復元
- 前回のウィンドウサイズ・最大化状態・分割位置の復元
